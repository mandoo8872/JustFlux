## 2024-05-03 - [File Upload Accessibility]
**Learning:** Custom file upload buttons that use visually hidden `<input type="file">` elements wrapped in `<label>` tags are generally inaccessible to keyboard and screen reader users by default. Wrapping the input does not automatically propagate keyboard events or provide descriptive semantics unless explicitly handled.
**Action:** When implementing custom file upload controls using `<label>`, always add `role="button"`, `tabIndex={0}`, an explicit `aria-label`, and an `onKeyDown` handler to listen for 'Enter' and 'Space' keys to programmatically trigger the underlying input's `click()` event. Also ensure to use i18n variables for tooltip and aria-labels instead of hardcoded text.
