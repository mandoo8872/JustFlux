## 2024-05-24 - File Upload Label Accessibility
**Learning:** When using a visually hidden `<input type="file">` wrapped in a `<label>` to create a custom styled upload button, the resulting element is not accessible to keyboard users. The `<label>` is not inherently focusable and doesn't trigger a 'click' event when 'Enter' or 'Space' is pressed, causing keyboard and screen reader users to be completely unable to upload files.
**Action:** Always add `tabIndex={0}`, `role="button"`, `aria-label`, and an explicit `onKeyDown` handler that triggers the internal input's `click()` method on 'Enter' or 'Space' for custom file inputs wrapped in labels.
